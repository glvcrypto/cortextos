# SOP: Content Publishing

**Owner:** Agent drafts, Aiden reviews and publishes
**Trigger:** As needed (blog posts, social content, email newsletters)
**Time estimate:** 1-2 hours per piece

---

## Blog Post Workflow

### 1. Topic Selection
- Pull from client's content plan (`/chino:content-plan <client>`)
- Or from keyword research (`/chino:keyword-research <client> <topic>`)
- Prioritize: low competition + high local relevance

### 2. Draft Creation
- Run `/content:blog` with topic, target keyword, client context
- Draft saves to `projects/<client>/deliverables/content/blog-draft-<slug>.md`
- Include: title, meta description, H2/H3 structure, internal links, CTA

### 3. Review Checklist
- [ ] No AI cliches or filler
- [ ] Canadian English (colour, behaviour, centre)
- [ ] Local Northern Ontario context where relevant
- [ ] Target keyword in title, H1, first paragraph, meta description
- [ ] Internal links to relevant service pages
- [ ] Clear CTA (contact form, phone, email)
- [ ] Brand voice match (check brand-guide.md)

### 4. Publishing
- [ ] Aiden publishes to Lovable site (or WordPress if applicable)
- [ ] Submit URL to GSC for indexing
- [ ] Share on social media (LinkedIn, Facebook, GBP post)
- [ ] Log in CONTEXT.md under content published

---

## Social Content Workflow

### 1. Create batch
- Run `/content:social` with client context
- Generate 5-7 posts per batch
- Save to `projects/<client>/deliverables/content/social-batch-YYYY-MM-DD.md`

### 2. Review and schedule
- Aiden reviews all posts
- Schedule via social platform or Buffer/Ayrshare (when MCP available)

---

## GBP Post Workflow

- Never put phone numbers or URLs in post body text
- Use CTA buttons only ("Call Now", "Learn More")
- Post 2-3 times per week per client
- Track performance in monthly report
