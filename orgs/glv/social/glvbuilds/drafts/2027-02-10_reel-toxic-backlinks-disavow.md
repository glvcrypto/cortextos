## Reel Script

**Hook (on screen):** "Your backlink profile probably has some junk in it. Here's the 3-step process to find it — and what to actually do."

**Step 1:** GSC → Links → Top linking sites. Look for patterns: gambling sites, payday loan directories, foreign-language link farms, exact-match anchor text coming from dozens of sites with no real content. One odd link is usually harmless. A cluster from the same network is a pattern.

**Step 2:** Before you disavow anything, check for a manual action. GSC → Security & Manual Actions → Manual Actions. If Google hasn't flagged your site, an unusual backlink profile alone is rarely causing ranking problems.

**Step 3:** If you do have a manual action from link spam, or a clear toxic pattern from old SEO work: create a plain text disavow file listing the domains (format: domain: example.com) and upload it via the Disavow Tool in Google Search Console.

**Ending:** The most common mistake is disavowing everything that looks odd. Irrelevant links are usually harmless. The disavow file is for confirmed harmful patterns — not routine housekeeping.

## LinkedIn Caption

I get asked about the disavow tool more than most local SEO topics. Usually right after someone discovers their backlink profile has some strange-looking links in it.

Here's the framework I use:

**Step 1: Check for a manual action first.**
GSC → Security & Manual Actions → Manual Actions. If Google has flagged your site for unnatural links, it will say so here explicitly. If there's no manual action, an unusual-looking backlink profile alone is unlikely to be causing ranking problems.

**Step 2: Look for patterns, not individual links.**
GSC → Links → Top linking sites. Red flags: exact-match anchor text from dozens of different sources, links from gambling/payday loan/adult sites that have no plausible connection to your business, foreign-language link farms, or a cluster of links that appeared all at once.

One link from a site you've never heard of? Probably harmless. A hundred links from the same suspicious network? That's a pattern worth acting on.

**Step 3: Disavow only what's confirmed harmful.**
The disavow file is a plain text document (domain: example.com) that you upload via Google Search Console. It tells Google to ignore those links when evaluating your site.

But: disavowing links that are actually neutral or positive can hurt your ranking. This is not a spring cleaning exercise. It's a last-resort tool for confirmed manual actions or sustained link spam from old SEO work.

Honest take: most local service businesses in competitive markets don't need to use the disavow tool at all. They need more good links, not fewer bad ones.
