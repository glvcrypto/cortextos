## Script

**Hook (on screen):**
"Your GSC Coverage report has errors. Here's which ones actually matter."

**Body:**

[Screen: GSC → Index → Pages]

Four statuses — only two require action.

**Error** → page not indexed, Google hit a problem. Fix these first.

**Warning** → indexed but something's off. Investigate.

**Valid with warning** → usually fine. Monitor, don't panic.

**Excluded** → Google chose not to index. Often correct.

**The one error I fix first every time:**
"Submitted URL not found (404)" — you submitted a URL to Google that returns a 404. That means you broke a page you told Google to index. Fix the redirect or restore the page.

**Second one to fix:**
"Redirect error" — page redirects to another redirect. Clean up the chain.

**Excluded: "Crawled, currently not indexed"**
This isn't an error — but it's a signal. Google crawled it and chose not to index it. Usually means thin content or duplicate.

Check it quarterly, don't ignore it.

**Ending:**
Four statuses. Two to fix. One to monitor.
