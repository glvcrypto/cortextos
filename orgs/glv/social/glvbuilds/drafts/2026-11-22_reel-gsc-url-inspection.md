## Reel Script (~35 seconds)

**[Hook — on screen + voiceover]**
"After I make any change to a page — any change — I run one check in Search Console before I call it done."

**[Step 1]**
Open GSC → URL Inspection → paste the page URL.
This pulls Google's live data on how it sees that specific page right now.

**[Step 2]**
Is it indexed?
If the status says "URL is not on Google" — that's your first problem. The page can't rank if Google can't see it.

**[Step 3]**
What canonical does Google see?
It should match the URL you submitted. If Google is choosing a different canonical — a redirect, an alternate version, a parameter URL — your ranking signals are going to the wrong page.

**[Step 4]**
Mobile usability.
Any issues flagged here affect how Google evaluates the page for mobile-first indexing.

**[Step 5]**
Hit Request Indexing.
Tells Google: "I just updated this page — come look at it again." Takes 24–72 hours to process.

**[Ending]**
One check. Every change. Catches indexation bugs before they cost you rankings.

## LinkedIn Caption (for sharing the reel)

Every time I update a page — title tag rewrite, new paragraph added, image swapped, schema change — I run one check before I move on.

GSC → URL Inspection → paste the URL.

It takes 30 seconds. It's caught indexation bugs that would have taken weeks to notice otherwise.

Here's what I check:

**Indexed or not.** If the status is "URL is not on Google," the page cannot rank. The update doesn't matter if Google can't see the page. Happens more often than you'd think — especially after a migration, a URL structure change, or an accidental noindex tag left in place.

**What canonical Google is choosing.** The canonical URL tells Google which version of a page to credit with ranking signals. It should match the URL I just edited. If Google is choosing a redirect target, a parameter URL, or an alternate version, my changes are going to the wrong page.

**Mobile usability.** Any mobile usability issues flagged here affect how the page is evaluated under mobile-first indexing. Better to catch them now than to watch rankings drift and wonder why.

**Request Indexing.** After checking the above — click it. This queues the page for a fresh crawl. It doesn't guarantee immediate re-ranking, but it tells Google the page was intentionally updated and is ready to be re-evaluated. Takes 24–72 hours.

This is distinct from the GSC weekly check (Oct 14), which is a broader impressions/CTR/coverage scan across the whole site. URL Inspection is a single-page, single-change confirmation step.

One check. Every change. Under a minute. Catches the kind of indexation problem that otherwise hides for weeks.
