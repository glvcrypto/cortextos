# 2027-02-24 | LinkedIn + Instagram | Reel (~30–35s) | CTA: open comment

**Date:** February 24, 2027
**Platform:** LinkedIn + Instagram
**Content type:** Reel — ~30–35 seconds
**CTA:** Open comment (no automation needed)
**Topic:** GBP post types — What's New vs. Offer vs. Event vs. Update (which one to use when)
**MEMORY cluster:** GBP post types comparison (new — add after session; distinct from Sep 30 post batching strategy)

---

## Script

**[Screen: GBP Manager → Posts tab showing post type selection options: What's New, Offer, Event, Update]**

**Hook (on screen + voiceover):**
"GBP has four post types. Most businesses use What's New for everything. Two of the four are almost never used — and they're the ones that drive calls."

---

**[Screen: What's New post editor]**

**Step 1 (on screen):**
**What's New** — general update. No deadline. Shows for 7 days.
Use for: tips, reminders, team news, anything without urgency.
It's the default — but it's the lowest-conversion type.

---

**[Screen: Offer post editor showing start date, end date, and coupon code fields]**

**Step 2 (on screen):**
**Offer** — has a start date, end date, and coupon code field.
Google shows a countdown on mobile listings when the end date is approaching.
Use for: seasonal promotions, booking incentives, limited-time service packages.
Deadline-driven posts drive more action than open-ended ones.

---

**[Screen: Event post editor showing event date and time fields]**

**Step 3 (on screen):**
**Event** — requires a specific date and time.
Shows on Google Maps in "upcoming events" for nearby searchers.
Use for: open houses, workshops, seasonal events, client info nights.
Most businesses never use this one.

---

**[Final screen: GBP Manager → Posts list showing one of each type]**

**Payoff (on screen):**
"What's New for general updates. Offer when you have a real deadline. Event when you have a date.
Using Offer instead of What's New for your next promotion is a 2-minute change."

CTA: "Comment below with what type your business should be using."
