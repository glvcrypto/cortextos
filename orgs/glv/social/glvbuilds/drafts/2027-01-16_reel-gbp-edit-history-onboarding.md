# GBP Edit History — First Thing I Check in Every New Client Onboarding

## Hook (on screen)

"Before I touch anything in a new client's GBP, I check one thing most agencies never look at: the edit history."

---

## Body (3–4 steps, one sentence each on screen)

**Step 1.** GBP Manager → Manage profile → View edit history. It shows every change ever made to the listing, who made it, and when.

**Step 2.** Third-party edits auto-applied by Google — hours changed, category swapped, description replaced. Most clients have no idea this happened.

**Step 3.** In about one in three accounts I inherit, something wrong was silently applied: a competitor's keyword in the name field, incorrect hours from a Maps suggestion, a category that no longer fits.

**Step 4.** If it's wrong, I correct it via "Suggest an edit" on the listing. Then I set up edit notifications so it doesn't happen again undetected.

---

## Ending

"Check edit history before you optimise. You might be building on top of a broken foundation."
