# Reel: The one question I ask before planning January

## Script (~30s)

**Hook (on screen):**
"Before I plan a single January task for a client, I ask them one question."

**Body:**

"What's bringing in the most calls right now?"

Not: what's ranking best.
Not: what's getting the most traffic.

The answer tells me exactly where January attention goes.

If it's the furnace repair page — we protect that ranking first, then expand.
If it's GBP direct search — we double down on profile completeness and GBP post consistency.
If they don't know — that's the first January task: set up GBP call tracking.

The most common Q1 mistake I see: spending January optimising a page that barely gets impressions, while ignoring the service that's already driving revenue.

Fix what's broken. But protect what's working first.

**CTA (on screen):**
"What question do you ask before you plan January? Drop it below."

## LinkedIn Caption (full)

**Before I plan January for any client, I ask one question: what's bringing in the most calls right now?**

Not what's ranking best. Not what's getting the most impressions.

The answer to that question tells me where 80% of January attention should go.

If it's the main service page — we protect that ranking before we touch anything else. Internal links tightened, GBP post scheduled, schema verified.

If it's GBP direct search (people searching the business name specifically) — we focus on profile completeness: Services tab filled out, Q&A seeded, photo cadence consistent.

If the client doesn't know — that's the first January task. Set up GBP call tracking and at least three months of baseline data before optimising anything based on feel.

The most common Q1 mistake: spending January rebuilding a page that gets 30 impressions a month while ignoring the service that's already driving revenue and could drive more.

Find what's working. Protect it. Then expand.

What question do you ask before planning January? I'm curious what other agencies use as a starting point.
