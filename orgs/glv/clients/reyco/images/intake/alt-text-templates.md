# Reyco Marine — Alt-Text Templates by Brand

**Prepared:** 2026-04-24
**Purpose:** Alt-text templates for Casey's 145-SKU image intake. Designer applies per image when staging brand folders.
**Format:** `{brand} {product_type} {model} {key_spec} — Reyco Marine Sault Ste. Marie`
**Length target:** 100–125 characters
**Regional keyword:** rotate naturally between "Sault Ste. Marie", "Northern Ontario", "Algoma"

---

## How to Use

Each brand has:
1. A **base template** — fill in `[model]` and `[key_spec]` from the product record
2. **Examples** showing the pattern in use
3. A **fallback** for images where model/spec is unavailable

Replace `[model]` with the consumer product name (not the model code — use the cleaned tag_product_name from the master CSV).
Replace `[key_spec]` with the single most buyer-relevant spec for that product type.

## Designer Notes

- Fill `[model]` from `tag_product_name` in the master CSV (cleaned consumer names, not model codes)
- Fill `[key_spec]` from the product record — use the single most buyer-relevant spec
- Rotate regional keywords: use "Sault Ste. Marie" for primary/hero images, "Northern Ontario" or "Algoma" for secondary/detail shots
- Do not use both "Sault Ste. Marie" AND "Northern Ontario" in the same alt-text — pick one per image
- Use the fallback template when product data is incomplete — a correct fallback beats an invented spec
- Verify final character count before applying — target 100–125 chars, minimum 60 chars
