1. WP Admin → Products → Categories → [Category name] → Description field → Save
2. Or bulk via WC REST API (write-scoped key required): `PUT /wp-json/wc/v3/products/categories/{id}` with `{"description": "..."}`
3. After all descriptions applied: RankMath → Titles & Meta → Products → Product Category → Meta Description = `%wc_category_description%`
