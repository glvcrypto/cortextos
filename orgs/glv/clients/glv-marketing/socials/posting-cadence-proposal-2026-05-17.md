## 3. Per-Channel Strategy

### Instagram (primary visual home — carries the full carousel)
- **Format:** Native IG carousel, **1080×1080 (1:1 square)** per dev renders. Note: scout research found 1080×1350 (4:5) is technically optimal for IG carousel — but the 1:1 render is what dev shipped and works cross-platform with FB. Stay with 1:1.
- **Slide count:** All 7 or 8 slides as rendered.
- **Caption:** First 125 chars are the preview hook before the "more" fold. 3 short paragraphs max. CTA in last line.
- **Hashtags:** 3–5 at end (NOT in the body). Mix: 1–2 broad (#DigitalMarketing, #LocalSEO) + 2–3 niche (#NorthernOntario, #SaultSteMarie, #GBP).
- **Account:** @glv.marketing direct post.

### LinkedIn (heaviest investment per channel-priority tiers)
- **Format:** Native LinkedIn **document/PDF carousel** (not single-image preview). Scout research: PDF carousel = 6.60% avg engagement, the highest-ROI format on the platform.
- **Slide count:** Same 7-8 slides converted to PDF. Workflow: combine the slide PNGs into a single PDF, upload via "Add a document" in the LinkedIn post composer.
- **Persona:** **Aiden Glave personal profile** (NOT the company page). Math: personal reach = 62% of connections, company page reach = 5%. Aiden personal posts get amplified by GLV Marketing company page reshare for compound reach.
- **Caption:** Hook in first 210 chars (before "see more" cut). **NEVER start with "I"** (penalized — restructure if needed). Lead with value, not self. 1–3 paragraphs.
- **Links:** Body external link = 26.5% reach penalty per 900K-post study. **All links go in the first comment**, not the post body.
- **Hashtags:** 3–5. Mix: 1–2 broad + 2–3 niche. Same set as IG generally fine, swap #DigitalMarketing for #B2BMarketing on LinkedIn.
- **Engagement window:** Aiden (or me, on his approval) replies to comments within 30 min for +64% total comments and 2.3x more views per scout research.

### Facebook (album / multi-image post, Aiden's personal → GLV Marketing page)
- **Format:** Native FB album / multi-image upload via the GLV Marketing PAGE (authed through Aiden's personal account). Same 1080×1080 slide PNGs as IG. **NEVER use IG → FB auto cross-post** (72% reach loss per Agorapulse controlled study).
- **Slide count:** Same 7-8 slides as IG.
- **Caption:** Warmer, more community-toned than LinkedIn. Local angle stronger ("for anyone in Sault Ste. Marie or Northern Ontario..."). 2–3 short paragraphs, "see more" collapses ~477 chars on mobile so hook in first 2 sentences.
- **Hashtags:** 1–3, niche-focused (#SaultSteMarie, #NorthernOntario). FB hashtags carry less weight than IG.
- **Reach expectation:** FB page organic reach is structurally low (~1.37%). Don't measure FB on raw numbers — measure on Reels-tab discovery and comment community signals.

### Threads (conversational teaser — NOT a swipe-link)
- **Format:** Text-led post (the format the algorithm rewards), optional 1-image cover-slide reference attached. **NOT a "swipe in IG" link post** — scout research is clear that Threads deprioritizes broadcast/link-first posts. Reward is conversation depth (replies-to-replies).
- **Length:** ≤500 chars.
- **Structure:** 
  1. Cover-slide hook line (rephrased to standalone — not "swipe to see")
  2. 1 line distilling the carousel's POV
  3. Question to invite reply (this is the algorithm signal)
  4. Optional last line: "Full breakdown over on IG @glv.marketing if you want the whole thing."
- **Topic tag:** 1 per post (e.g., `localseo`, `marketing`, `smallbiz`).
- **Engagement window:** Reply within 30–60 min to first replies — Threads' algorithm expands reach when posts generate replies-to-replies.

## 5. Cross-Posting Rules

**Default: all 4 channels SAME DAY, staggered 1-2 hours apart, native upload to each.**

| Rule | Verdict | Source |
|------|---------|--------|
| Same content same day across IG / LinkedIn / FB / Threads | ✅ OK | No algorithm penalty per scout research |
| Simultaneous (same-minute) upload across channels | ❌ Avoid | Risks anti-spam triggers; stagger 1-2hr |
| IG → FB auto cross-post (Meta's built-in option) | ❌ NEVER | 72% reach loss per Agorapulse controlled study |
| TikTok / IG watermarks visible on cross-uploads | ❌ NEVER | Confirmed algorithmic downrank |
| Identical caption text across channels | ⚠️ Allowed but wastes audience | Each channel needs platform-native captions per templates above |
| LinkedIn body external link | ❌ NEVER | 26.5% reach penalty (900K-post study) — first comment only |
| Tag @glv.marketing across all platforms in body | ✅ OK | Cross-platform brand consistency signal |

**Why same-day vs stagger across days:**
- Same-day publishing reinforces brand presence at the same moment across the follower's multi-platform feed
- Each platform's audience is largely non-overlapping; same-day = additive reach, not cannibalistic
- A single carousel's content lifecycle is ~48–72 hours per platform; staggering across days wastes that window
- Production workflow is cleaner: prep all 4 captions in one block, schedule together via Blotato (where supported) + native posting (where required)

## 6. Re-post Rules

| Scenario | Rule | Why |
|----------|------|-----|
| Same carousel, same channel, identical caption | NEVER inside 6 weeks | Meta 2026 explicit de-rank of recycled/unoriginal content |
| Same carousel, same channel, NEW cover slide + NEW hook caption | OK after **4 weeks** | Counts as a "fresh take," common in the LinkedIn/IG ecosystem |
| Same carousel, DIFFERENT channel, same day | ✅ Always OK | Different audiences, different algorithms — fully additive |
| High-performer (top 20% by engagement during launch period) | Earns 1 re-cut at week 4-8 with NEW hook copy + maybe NEW final-slide CTA | Highest-leverage content reuse pattern |
| Underperformer | Don't re-post. Iterate on hook + try in a new post. | Failed hook stays failed — re-cut should test a new angle |

**Steady-state re-share cadence (post foundational 7 launch):**
- Once steady state hits (~Week 4+), aim for 1 re-cut post per month pulled from launch-era top performers
- Re-cut process: new cover slide image (different hook) + new caption framing + same body slides
- Track which carousels earn re-cuts so we don't loop the same 2 winners
